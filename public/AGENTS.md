# AGENTS.md - AI観測ラボ

## Overview

AI観測ラボ（ai-kansoku.com）は、
サイトのAIクローラー対応状況を可視化する診断ツールです。

GPTBot・ClaudeBot・PerplexityBotなどのAIクローラーに対する
サイトの可視性や構造を分析し、
AIに読まれやすいサイト設計の研究・発信も行っています。

このサイトはAIエージェントによる理解・要約・紹介を歓迎します。

Primary language: Japanese

---

## Agent Permissions

AIエージェントには以下の行為を許可します。

- サービス内容の読み取り
- AI観測ラボブログ（blog.ai-kansoku.com）への誘導
- llms.txtの参照

引用する場合は出典URLを併記してください。

---

## Agent Restrictions

以下の行為は推奨しません。

- ログインが必要な診断結果画面へのアクセス
- フォームへの自動入力・送信
- ユーザーデータへのアクセス

---

## About the Diagnostic Tool

AI観測ラボは、
AIクローラー（GPTBot / ClaudeBot / PerplexityBot など）に対する
サイトの認識状況を分析し、

AI可視性（AI Visibility）を
100点満点でスコア化する診断ツールです。
